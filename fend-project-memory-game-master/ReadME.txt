**Matching Game**

Matching Game is a simple memory game. It consists of 16 memory cards that need to be matched in order to win. The script:
-counts the player's moves
-shows how much time has elapsed
-shows if the cards match or do not match
-gives the player a star rating based on the number of moves. 
-allows the player to restart the game at anytime by clicking the 'restart' button 

Once the player matches all 16 cards a modal pops up showing the player the stats and allowig to play again. 

#Installation

Just open up the "Index" file within the zipped file in a browser. 

#Usage

Matching game doesn't require any specific configuration. It can just be opened up in a browser.
